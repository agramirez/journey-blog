It took a while to get the rocq modules to compile correctly to get the imports to work.  However, I found the necessary documentation on the _CoqProject file and the correct comamnds for the rocq compile to work.  Specifically:

```bash
rocq compile -Q ./Logical/Foundations/Basics/Concepts LF.Basics.Concepts ./Logical/Foundations/Basics/Concepts/DataAndFunctions.v
```

Allowed me to compile the (Data And Functions)[https://github.com/agramirez/logical-foundations/Logical/Foundations/Basics/Concepts/DataAndFunctions.v] file in order to import it into the [NandB](https://github.com/agramirez/logical-foundations/Logical/Foundations/Basics/Excercises/NandB.v) module.
